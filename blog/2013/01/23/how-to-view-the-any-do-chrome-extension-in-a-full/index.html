How to View the Any.DO Chrome Extension in a Full Browser Window

Jan 23rd, 2013 | Comments

I complained (okay, more like whined…) in December that Any.DO was rejecting user requests for auto-sync with Google Tasks. I’ve since come to use Any.DO full time, after trying it a while ago, loving it, being distracted by Astrid and Wunderlist, and coming back for the simultaneously unique and intuitive user interface.

I signed up a while ago to be notified when Any.DO launches their web interface. While their Chrome extension is robust, I don’t really enjoy poking around in a pop-up that truncates most of my task names. Both Astrid and Wunderlist already have web interfaces, but I’ve always preferred Any.DO.

That’s why I decided to poke around in the code behind Any.DO’s Chrome extension. I’m not an expert, but I’m knowledgeable enough to go in and pull out the Chrome address for the extension itself. If you have already installed the extension, this link will open it in a full browser tab of its own, thus eliminating the truncated-tasks problem.

UPDATE: Thanks to a helpful comment from DariusXL I’ve fixed the link in the previous paragraph and it should work as expected. Cheers to jonS for posting more explicit directions for this on his Kinja.

It’s not a proper web interface, since it doesn’t work unless the extension is installed, but feels close enough to hold me over until they release the real deal.
